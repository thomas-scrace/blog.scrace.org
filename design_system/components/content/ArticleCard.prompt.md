A post preview for index/archive lists — eyebrow meta, serif title (gilt on hover), standfirst, tag row.

```jsx
<ArticleCard
  kicker="Essay"
  date="9 June 2026"
  readingTime="7 min"
  title="On Stillness & Attention"
  standfirst="What we lose when we never let a thought finish."
  tags={["essays", "reading"]}
/>
```

Default is a list row with a bottom hairline. Set `bordered` for a standalone card that fills on hover (use for featured posts).
